Overview

Welcome to CSCI 8360: Data Science Practicum.

In this class, you will design and implement large-scale machine learning solutions to real-world problems. You'll work in teams to design machine learning pipelines, customize and scale algorithms, and ensure a smooth software engineering process.

This class IS...

  • Purely project-based: five 2-week projects, and a 4-week final project
  • An opportunity to explore your own solutions and apply your unique talents, free of specific step-by-step project requirements aside from maximally satisfying a machine learning goal
  • A practicum!

This class is NOT...

  • An introduction to machine learning (CSCI 4380/6380 is a prerequisite)
  • An introduction to data science (CSCI 4360/6360 is a prerequisite)
  • An introduction to software engineering (take CSCI 4050/6050)
